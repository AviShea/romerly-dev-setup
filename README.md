# Romerly – Developer Setup Package

This package contains everything needed to understand, run, and extend the Romerly website-first product.

## Stack
- Next.js (React + TypeScript)
- Supabase (Auth, Postgres, RLS)
- Kiwi (Flights – redirect model)
- OpenAI (AI Planner)
- Vercel (Hosting)

## Start Here
1. Read ARCHITECTURE.md
2. Review /supabase/schema.sql
3. Copy .env.example → .env.local
4. Run `npm install && npm run dev`
