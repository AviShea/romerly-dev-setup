# API Routes

## Flights
GET /api/flights/search → builds Kiwi redirect URL
POST /api/flights/add → save booked flight to trip

## Trips
POST /api/trips/create
GET /api/trips/:id

## Voting
POST /api/votes/create
POST /api/votes/:id/cast

## AI Planner
POST /api/ai/itinerary

## Safety
POST /api/safety/sos
